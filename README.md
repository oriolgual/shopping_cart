# ShoppingCart

## Installation

Add this line to your application's Gemfile:

```bash
git clone git@github.com:oriolgual/shopping_cart.git
```

And then execute:

    $ cd shopping_cart
    $ bundle install

## Usage

    $ irb
    $ require 'shopping_cart'

## Test suite

    $ bundle exec rake

or:

    $ bundle exec guard
